# Issue Template for API Integration Issues

## 📋 Preços de Submissão

Antes de criar um novo issue, verifique se já existe uma questão similar que seja aberta ou fechada.

### 📌 Informação do Problema

Por favor inclua a seguinte informação:

- **Provedor da API:** (Ex: EvolutionGO, Z-API, UZAPI, etc)
- **Plataforma:** WhatsApp / Instagram / Facebook / TikTok / Twitter
- **Ambiente:** Sandbox ou Produtiva

### 🔧 Detalhes do Problema

Explique claramente o que você está tentando fazer e o que está acontecendo.

### 💻 Código de Reprodução

Se aplicável, inclua um trecho de código mínimo que reproduza o problema.

```typescript
// Exemplo:
await whatsappApi.sendMessage({ to: 'number', message: 'Hello' });
```

Cole aqui os logs ou mensagens de erro completas.
